# react native image picker and acces camera


## Installation

1. Clone the repository:

   ```bash
   git clone https://github.com/palveeen22/react-native-image-picker-camera.git

2. Open the repository:

   ```bash
   cd react-native-image-picker-camera
   npm install

3. Run the App:

   ```bash
   open your simulator/emulator
   npm run Android
   npm run ios
